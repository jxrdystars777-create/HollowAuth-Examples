# HollowAuth C# Example

Example oficial de integración de `HollowAuth` para aplicaciones desarrolladas en `C#`.

> [!IMPORTANT]
> HollowAuth continúa siendo un proyecto personal en fase `Beta`.
> Algunas funciones o métodos pueden cambiar durante el desarrollo.

## Estado

`Disponible`

## Requisitos

- Visual Studio 2022
- .NET compatible
- Credenciales de una aplicación de HollowAuth

> [!WARNING]
> No publiques tu `Application Secret`, licencias reales ni credenciales privadas.
