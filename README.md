Repositorio de examples para integrar `HollowAuth` en diferentes lenguajes y aplicaciones.

> [!IMPORTANT]
> `HollowAuth` es un proyecto personal que actualmente se encuentra en fase `Beta`.
> Algunas funciones, métodos y respuestas de la API pueden cambiar durante el desarrollo.

## Compatibilidad

| Lenguaje | Estado |
|---|---|
| `C#` | Disponible |
| `C++` | Disponible |
| `Python` | Próximamente |
| `JavaScript` | Próximamente |
| Otros | Planeados |

## Examples disponibles

### C#

Example para aplicaciones de escritorio con Windows Forms.

```text
/csharp
```

### C++

Example para aplicaciones de escritorio en Windows.

```text
/cpp
```

> [!TIP]
> Los examples de `C#` y `C++` ya están disponibles y pueden utilizarse como referencia para integrar HollowAuth.

## Próximamente

La compatibilidad con `Python`, `JavaScript` y otros lenguajes será agregada progresivamente.

## Seguridad

> [!WARNING]
> Nunca publiques tu `Application Secret`, licencias, tokens, contraseñas o credenciales privadas.

Usa valores de ejemplo como:

```text
YOUR_APP_NAME
YOUR_APP_ID
YOUR_APP_SECRET
YOUR_LICENSE
```

## Reportar errores

Puedes abrir un `Issue` indicando:

- Lenguaje utilizado.
- Descripción del error.
- Pasos para reproducirlo.
- Capturas, cuando sean necesarias.

> [!CAUTION]
> Elimina cualquier dato privado antes de publicar un reporte.
