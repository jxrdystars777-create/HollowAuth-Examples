Repositorio oficial de examples para integrar `HollowAuth` en diferentes lenguajes y aplicaciones.

> [!IMPORTANT]
> `HollowAuth` es un proyecto personal que actualmente se encuentra en fase `Beta`.
> Algunas funciones, métodos, SDKs y respuestas de la API pueden cambiar durante el desarrollo.

## Compatibilidad

| Lenguaje | Estado |
|---|---|
| `C#` | Disponible |
| `C++` | Disponible |
| `Python` | Próximamente |
| `JavaScript` | Próximamente |
| Otros lenguajes | Planeados |

## Examples disponibles

### C#

➡️ [Ver example de C#](./csharp/)

### C++

Example oficial para aplicaciones de escritorio desarrolladas con `C++`.

➡️ [Ver example de C++](./cpp/)

## Próximamente

La compatibilidad con `Python`, `JavaScript` y otros lenguajes será agregada progresivamente.

## Seguridad

> [!WARNING]
> Nunca publiques tu `Application Secret`, licencias reales, tokens, contraseñas o credenciales privadas.

Utiliza valores de ejemplo como:

```text
YOUR_APP_NAME
YOUR_APP_ID
YOUR_APP_SECRET
YOUR_LICENSE
```

## Soporte

Para solicitar ayuda, reportar errores o enviar sugerencias:

- **Discord:** [Servidor oficial de HollowAuth](https://discord.gg/AS2JtmUEvg)
- **GitHub:** Abre un `Issue` dentro de este repositorio.

> [!NOTE]
> Cuando tengamos un dominio oficial, el correo será reemplazado por una dirección profesional como `support@hollowauth.com`.

> [!CAUTION]
> No compartas licencias, contraseñas, tokens o secretos dentro de canales públicos.

## Reportar errores

Al abrir un `Issue`, incluye:

- Lenguaje utilizado.
- Descripción del problema.
- Pasos para reproducirlo.
- Mensaje de error.
- Capturas de pantalla, cuando sean necesarias.
